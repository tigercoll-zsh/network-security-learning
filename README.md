## 🚀 项目介绍

这不是一本枯燥的教科书，而是一份**红队实战手册**。

大多数网安教程都是"先讲一堆理论，再做个小实验"，结果学完了一堆 OSI 七层模型，连个简单的 SQL 注入都跑不通。

**本项目反其道而行之：**

1. **场景驱动**：每一天都从一个真实的"痛点场景"开始（比如：网站被拖库了、密码被爆破了）。
2. **实战优先**：先动手把漏洞复现出来，拿到 Shell，感受到快感，然后再去补理论。
3. **闯关制度**：没有"大概懂了"，只有"做出来了"。每天都有明确的闯关条件。

## 📅 三阶段进化论

我们把 90 天分为三个阶段，模拟一个黑客的成长路径：

### 第一阶段：🔨 筑基（Day 1-30）
> **目标**：从"我电脑能上网"进化到"我能看到别人电脑的数据"。

- **Week 1**: 破冰行动 —— 第一次黑客体验
- **Week 2**: 侦察 —— 信息收集与扫描
- **Week 3**: Web 入门 —— SQL 注入与数据库
- **Week 4**: 客户端攻击 —— XSS 与文件上传
- **Week 5**: 阶段考核 —— 拿下 DVWA

### 第二阶段：⚔️ 攻坚（Day 31-60）
> **目标**：从"会用工具"进化到"理解原理 + 能绕过防护"。

- **Week 6**: SQL 注入进阶 —— 盲注与 WAF 绕过
- **Week 7**: XSS 进阶 —— 编码混淆与 CSRF
- **Week 8**: 逻辑漏洞 —— 越权与支付漏洞
- **Week 9**: 系统提权 —— Linux/Windows 提权
- **Week 10**: 阶段考核 —— 打穿 HackTheBox 靶机

### 第三阶段：🏆 实战演练（Day 61-90）
> **目标**：从"打靶机"进化到"打真实场景"（内网/红蓝对抗）。

- **Week 11-12**: 内网渗透 —— 域渗透与隧道代理
- **Week 13**: 红蓝对抗 —— 防御视角与免杀
- **Week 14**: 综合实战 —— 完整渗透测试项目
- **Week 15**: 终极考核 —— 报告撰写与毕业答辩
