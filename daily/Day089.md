### 任务 1：信息收集（90 分钟）

**必做步骤**：

```bash
# 1. 主机发现
nmap -sn <target_range>

# 2. 全端口扫描
nmap -sV -sC -p- -oA scans/full <target_ip>

# 3. 服务枚举
# 根据发现的服务进行针对性枚举

# 4. 记录所有发现
```

|------|------|------|------|

### 任务 3：权限提升（90 分钟）

**必做步骤**：

```bash
# Linux

find / -perm -4000 2>/dev/null
cat /etc/crontab

# Windows
whoami /priv
systeminfo
```

## 📖 底层补课（做完实验再看）

### 考核自评标准

| 维度 | 优秀 | 良好 | 需改进 |
|------|------|------|--------|
| 信息收集 | 全面、有条理 | 基本完整 | 遗漏重要信息 |
| 漏洞利用 | 高效、多种尝试 | 能完成 | 依赖提示 |
| 提权能力 | 独立完成 | 少量查阅 | 大量依赖外部 |
| 记录习惯 | 详细、专业 | 基本记录 | 缺少记录 |

### 常见卡点

| 卡点 | 可能原因 | 建议 |
|------|----------|------|
| 找不到入口 | 枚举不够 | 重新仔细枚举每个服务 |
| 利用失败 | 版本/配置不同 | 确认版本，尝试其他方法 |
| 提权无思路 | 自动化工具依赖 | 手动枚举，回顾所学 |
