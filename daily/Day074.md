**推荐靶场**：

- HackTheBox: Active Directory 相关靶机
- 自建 AD 环境

**攻击流程**：

```
┌─────────────────────────────────────────────────────────┐
│ 1. 外网信息收集                                          │
│    └── Nmap 扫描 → 发现 Web 服务                        │

│    └── Web 漏洞利用 → 获取 Webshell                     │
│                                                          │
│ 3. 建立隧道                                              │

│                                                          │
│ 4. 内网信息收集                                          │
│    └── 发现域环境 → 枚举用户和机器                      │
│                                                          │
│ 5. 横向移动                                              │
│    └── PTH/Kerberoasting → 获取更多权限                │
│                                                          │
│ 6. 域控攻击                                              │
│    └── DCSync → 获取所有 Hash                           │

│    └── 建立后门 → 清理痕迹                              │
└─────────────────────────────────────────────────────────┘
```
