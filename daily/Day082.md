### 任务 2：环境准备（60 分钟）

**选项 A：HackTheBox**

```bash
# 注册账户并连接 VPN
# 下载 VPN 配置文件
sudo openvpn lab_username.ovpn

# 验证连接
ping 10.10.10.x

# 选择一台机器开始
```

**选项 B：VulnHub 本地环境**

```bash
# 下载 VM 镜像
# 推荐：DC-1, DC-2, DC-3 系列（AD 入门）

# 导入 VMware/VirtualBox
# 配置网络（NAT 或 Host-Only）

# 确认机器启动
nmap -sn 192.168.x.0/24
```

**选项 C：自建 AD 环境**

```powershell
# Windows Server 安装 AD DS
Install-WindowsFeature -Name AD-Domain-Services -IncludeManagementTools

# 提升为域控
Install-ADDSForest -DomainName "corp.local"

# 创建测试用户和组
New-ADUser -Name "TestUser" -Enabled $true -AccountPassword (ConvertTo-SecureString "Password123" -AsPlainText -Force)
```

## 📖 底层补课（做完实验再看）

### 靶场选择原则

1. **匹配水平**：太简单没挑战，太难打击信心
2. **覆盖技术**：选择能覆盖所学技术的靶场
3. **有参考**：有 Writeup 可以在卡住时参考
4. **可复现**：确保环境稳定可用

### 靶场使用建议

| 建议 | 说明 |
|------|------|
| 先独立尝试 | 不要一开始就看 Writeup |
| 记录过程 | 每一步都要记录 |
| 卡住求助 | 2 小时没进展可以看提示 |
| 复盘总结 | 完成后写完整总结 |

## ⚠️ 避坑指南

| 坑点 | 症状 | 解决方案 |
|------|------|----------|

| VPN 断开 | HTB 连不上 | 重新连接/换区域 |
| 选错难度 | 卡很久无进展 | 换简单的机器 |
