## 闯关任务

### 阶段 1：目标确认（10 分钟）

1. 确认 Metasploitable2 已启动
2. 获取目标 IP 地址
3. 确认网络连通性

```bash
# Ping 测试
ping 192.168.56.102

# 确认你的 Kali IP
ip addr
```

---

### 阶段 2：主机发现（15 分钟）

扫描整个网段，确认存活主机：

```bash
sudo nmap -sn 192.168.56.0/24
```

记录：
- 发现了多少台主机？
- 除了 Metasploitable2 还有什么？

---

### 阶段 3：端口扫描（30 分钟）

**快速扫描**：
```bash
nmap -T4 192.168.56.102
```

**全端口扫描**：
```bash
sudo nmap -sS -p- -T4 192.168.56.102 -oN ports.txt
```

**服务版本识别**：
```bash
sudo nmap -sV -sC -p 21,22,23,25,80,139,445,3306 192.168.56.102 -oN services.txt
```

记录：
- 开放了多少个端口？
- 每个端口运行什么服务？
- 服务版本是什么？

---

### 阶段 4：漏洞评估（30 分钟）

**Nmap 漏洞脚本**：
```bash
sudo nmap --script vuln 192.168.56.102 -oN vulns.txt
```

**搜索已知漏洞**：
```bash
searchsploit vsftpd 2.3.4
searchsploit samba 3.0
searchsploit apache 2.2
```

记录：
- 发现了哪些潜在漏洞？
- CVE 编号是什么？
- 严重性如何？

### Week 2 命令速查

```bash

site:example.com filetype:pdf

# WHOIS
whois example.com

# DNS 查询
dig example.com ANY
nslookup -type=MX example.com
```
