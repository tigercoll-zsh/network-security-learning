## 🔥 今日痛点场景

> **场景**：某公司员工用 `123456` 作为密码。黑客用密码列表自动尝试登录，5 分钟后成功进入系统。
>
> **问题**：黑客是怎么"自动猜密码"的？
>
> **今天的任务**：使用 Burp Intruder 加载密码字典，自动化爆破 DVWA 登录。

---

## 🎯 今日目标

- [ ] 将登录请求发送到 Intruder
- [ ] 设置攻击位置（标记密码字段）
- [ ] 加载密码字典
- [ ] 分析结果，找到正确密码

**闯关条件**：成功爆破出 DVWA 的 admin 密码。

---

## ⚔️ 实战任务（先动手！）

### 任务 1：抓取 Brute Force 页面的登录请求（10 分钟）

1. 访问 DVWA → Brute Force 模块
2. 开启 Burp Intercept
3. 输入 admin / wrongpassword，点击 Login
4. 右键 → Send to Intruder

---

### 任务 2：配置 Intruder（15 分钟）

**Positions 标签**：
1. 点击 Clear §
2. 选中 `wrongpassword`，点击 Add §
3. 结果：`password=§wrongpassword§`

---

### 任务 3：加载字典（10 分钟）

**Payloads 标签**：

手动添加几个密码：
```
123456
password
admin
qwerty
```

或加载 Kali 自带字典：
```bash
head -1000 /usr/share/wordlists/rockyou.txt > ~/passwords.txt
```

---

### 任务 4：开始攻击（20 分钟）

1. 点击 Start attack
2. 观察 **Length** 列
3. 长度不一样的就是正确密码！

## 📖 底层补课

### 什么是暴力破解？

就像撞锁——拿一串钥匙一把一把试，直到打开为止。

### Intruder 攻击模式

| 模式 | 用途 |
|------|------|
| Sniper | 单个位置逐个尝试 |
| Cluster Bomb | 多位置全排列 |

### 为什么看响应长度？

登录成功的页面内容更多，所以响应更长。
