## 考核任务

### 阶段 1：环境准备（10 分钟）

**确认环境**：

1. 启动 Kali Linux
2. 启动 DVWA 靶机
3. 确认网络连通
4. 设置 DVWA Security Level 为 **Medium**

```bash

ping <DVWA_IP>
```

**重要**：今天的考核全部在 Medium 级别进行，不是 Low！

## 如果卡住了

**不要立即放弃！**

问自己这些问题：

1. **SQL 注入卡住**：
   - 是字符型还是数字型？
   - Medium 级别用了什么防护？
   - 有没有尝试大小写或编码？

2. **XSS 卡住**：
   - 过滤器过滤的是什么？
   - 试过大小写混淆吗？
   - 试过其他事件属性吗？

3. **文件上传卡住**：
   - 检查的是后缀还是 MIME？
   - Burp 里改了什么？
   - 文件内容有没有问题？
